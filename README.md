dispeRse
========

This repository is an in developemnt R package intended to [generate new models for detecting phylogenetic signals of endemism and dispersal](http://www.nescent.org/science/awards_summary.php?id=481) and is supported by two [Short-Term Visiting Scholarships](http://www.nescent.org/dir/visiting_scholar.php?type=Short-term%20Visitor) from [NESCent](http://www.nescent.org/) (to [Laura Soul](https://github.com/laurasoul) and [Graeme Lloyd](https://github.com/graemetlloyd)).

More to follow.
